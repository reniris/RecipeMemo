提督業プラグイン レシピメモ表示

艦これWikiの開発レシピとユーザー定義のレシピを表示できます
ユーザー定義のレシピは一度起動すると出てくるUserRecipe.xamlを改変することで変更できます
デフォルトのユーザーレシピはあきつ丸レシピのみです
